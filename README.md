# Explorando o Linux Ubuntu

Projeto desenvolvido para a disciplina de **Experiência Criativa** da **Pontifícia Universidade Católica do Paraná (PUCPR)**.

## Descrição
Aplicativo educativo desenvolvido em **Processing** que explora a história do Linux Ubuntu, sua comunicação com o kernel e seu impacto social. A ferramenta utiliza uma interface interativa com perguntas e descrições para proporcionar um aprendizado imersivo e tátil sobre a distribuição.
